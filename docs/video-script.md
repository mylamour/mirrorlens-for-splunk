# Video Script

## Three-Minute Flow

1. Problem: security teams have Splunk data, but incident investigation and
   detection validation are still too manual.
2. Demo data: a sanitized credential leak, web exploit, lateral movement, and
   detection-gap scenario is loaded into Splunk.
3. Investigate: the agent uses Splunk MCP Server to discover hosts, users,
   sourcetypes, and timeline evidence.
4. Explain: the agent explains the SPL and evidence in analyst-friendly terms.
5. Recommend: the agent proposes detection improvements and dry-run response
   actions without touching production systems.
6. Value: faster investigation, grounded evidence, safer response, and clearer
   detection coverage.
