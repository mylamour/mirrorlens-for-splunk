# MirrorLens for Splunk

MirrorLens for Splunk is an open-source Splunk Agentic Ops Hackathon demo.
It uses sanitized attack-chain data, Splunk MCP Server, and a lightweight AI
agent workflow to investigate a security incident, explain evidence, identify
detection gaps, and propose safe dry-run response actions.

## Open Source Boundary

This repository contains only the public hackathon demo harness:

- Splunk ingest scripts and example configuration
- Sanitized demo data
- MCP agent workflow
- SPL query templates
- CLI or lightweight UI entrypoints
- README, architecture diagram, video script, and tests

It does not contain commercial X2 product source code, production deployment
logic, customer data, real logs, secrets, license material, or private model
routing.

## Hackathon Target

- Track: Security
- Bonus prize target: Best Use of Splunk MCP Server
- Project phrase: an open-source Splunk MCP agent demo for automated security
  investigation and detection validation.
